Upgrading Laravel 4 to Laravel 5.3

Sep 15, 2016 | Coding

The tech stack behind my startup is Laravel 4. Although the journey has pretty much finished, I would still like to upgrade it to the latest Laravel which is 5.3 to understand more about the latest version. The steps that I take is following the upgrade guide from 4.2 upgrade to 5.0 then 5.1->5.2 and then end 5.3->5.2.

I have try to go straight to 5.3 initially but realise there is so much things I have missed so I actually decided to upgrade it one by one at the end. Remember to go through the deprecations part since the section is small and easy to miss. Two hip up I have after the upgrade is lists() no longer a array but just a collection. In order to get back the whole array, add ->all() behind.

Lastly, lists has been deprecated and you need to use pluck instead. So

pluck()->all()

is what you need to use in the future.
