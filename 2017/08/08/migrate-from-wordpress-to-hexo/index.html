Migrate from wordpress to hexo

Aug 8, 2017

Moving away from wordpress is not something which I expected. I was typing my last blog about workflow engine and previewing it online but somehow the layout just not correct. I have double check a lot of things but it just wrong. All i want is something simple to display my blog content and easy to run so I decided to look for other option.

I play with Ghost before so I jump straight to their website and check out what’s new. Ghost is my first preferences since they got a clean, easy to use UI and some nice themes. The only down side is their hosting fee is quite expensive and I really don’t want to spend anytime on managing my server anymore. Suddenly I remember I have heard someone talking about using Github as their static site hosting and thinking it might be a good option for me. After some research, I have locked down my option in Hexo. (https://hexo.io/)
